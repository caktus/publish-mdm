Kiosk Mode

The Kiosk Custom Launcher toggle below replaces the Android launcher with a dedicated kiosk launcher, putting the whole device in kiosk mode. The customization settings beneath it apply in this mode, or when any app has install type Kiosk (configurable in the Applications section). Note: when the custom launcher is enabled, any app set to Kiosk install type is automatically treated as Force Installed — these two modes are mutually exclusive in Android Enterprise.

Getting Started
===============

This section is for onboarding and first-run setup. Start here if you are new to
Publish MDM and want quick, practical guides for your first successful workflow.

.. note::

   The guides in this section assume your organization is using the default and
   most common MDM integration, **Google's Android Management API.** If your organization
   is using **TinyMDM,** refer to the
   `TinyMDM documentation <https://www.tinymdm.net/help-resources/>`_ for help getting started.
